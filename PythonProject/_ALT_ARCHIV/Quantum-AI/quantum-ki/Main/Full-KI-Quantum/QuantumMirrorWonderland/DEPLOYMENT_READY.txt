   git init
   git add .
   git commit -m "🌀 Quantum Mirror Wonderland v1.0"
   git remote add origin https://github.com/YOUR_USERNAME/QuantumMirrorWonderland.git
   git branch -M main
   git push -u origin main

SCHRITT 3: Auf Render deployen
   1. Gehe zu https://render.com
   2. Sign up with GitHub
   3. New Web Service
   4. Wähle: QuantumMirrorWonderland Repository
   5. Configure:
      - Name: quantum-mirror-wonderland
      - Environment: Python 3
      - Build: pip install -r backend/requirements.txt
      - Start: gunicorn --chdir . app:app --bind 0.0.0.0:$PORT
   6. Deploy Button klicken
   7. FERTIG! App läuft auf: https://quantum-mirror-wonderland.onrender.com
