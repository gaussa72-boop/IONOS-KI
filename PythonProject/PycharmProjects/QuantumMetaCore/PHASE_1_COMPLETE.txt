🚀 SCHNELLSTART
═════════════════════════════════════════════════════════════════════════════

LOKAL (macOS/Linux):

1. Abhängigkeiten installieren:
   pip install -r requirements.txt

2. Environment Setup:
   cp .env.example .env

3. Entwicklungs-Server starten:
   python wsgi.py

   → App verfügbar unter: http://localhost:5000


MIT DOCKER:

1. Services starten:
   docker-compose up -d

2. Status prüfen:
   docker-compose ps

3. Logs anschauen:
   docker-compose logs -f backend

   → App verfügbar unter: http://localhost


AUTOMATISCHES SETUP (macOS):

   bash install/install_mac.sh

💾 DATENBANK-SCHEMA
═════════════════════════════════════════════════════════════════════════════

USERS TABLE:
  id          | INTEGER PRIMARY KEY
  username    | STRING (unique)
  email       | STRING (unique)
  password    | BINARY (hashed)
  created_at  | DATETIME

PROFILES TABLE:
  id                  | INTEGER PRIMARY KEY
  user_id             | FOREIGN KEY
  tone                | STRING (mystisch/strategisch/kreativ)
  depth               | STRING (einfach/standard/erweitert)
  level               | INTEGER (1-10)
  interaction_count   | INTEGER

TRAINING_DATA TABLE:
  id              | INTEGER PRIMARY KEY
  user_id         | FOREIGN KEY
  input_text      | TEXT
  response_text   | TEXT
  module          | STRING
  created_at      | DATETIME

GENERATED_GAMES TABLE:
  id              | INTEGER PRIMARY KEY
  user_id         | FOREIGN KEY
  game_id         | STRING (unique)
  idea            | TEXT
  genre           | STRING
  plays           | INTEGER
  rating          | FLOAT
