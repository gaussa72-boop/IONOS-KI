## ⚠️ HÄUFIGE FEHLER & LÖSUNGEN

**"ModuleNotFoundError: No module named 'flask'"**
```bash
pip3 install flask werkzeug
```

**"Port 5000 already in use"**
```bash
# Ändere Port in app_unified.py (Zeile ~400):
app.run(host='0.0.0.0', port=5001, debug=True)
```

**"Template not found"**
```bash
# Überprüfe, ob templates/ Ordner existiert:
ls templates/
```

**Keine Ausgabe beim Start**
```bash
# Nutze direkt:
python3 RUN.py
```
